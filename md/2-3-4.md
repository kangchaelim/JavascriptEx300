비교 연산자는 두 변수(또는 데이터)의 값을 서로 비교하는데 사용됩니다.

비교 연산자의 종류
| ==           | 같다 |
| --- | ------------ |
| ===    | 값과 데이터 형이 같다             |
| !=     | 다르다                      |
| !==    | 값과 데이터형이 다르다             |
| >      | 크다                       |
| <      | 작다                       |
| >=     | 크거나 같다                   |
| <=     | 작거나 같다                   |
